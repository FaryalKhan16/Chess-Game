# Chess Game | Java

A fully functional Chess Game built in Java with a graphical user interface, supporting two-player gameplay with all standard chess rules.


## Features:

. Complete Chess Logic - All standard chess moves implemented

. Graphical Interface - Interactive GUI with piece images

. Two-Player Mode - Play with a friend on the same computer

. Move Validation - Legal move checking for all pieces

. Turn Management - Automatic turn switching between players

. Invalid Move Prevention - Prevents illegal moves

. Piece Selection Highlighting - Visual feedback for selected pieces

. Check/Checkmate Detection - Game state monitoring

## Tech Stack:

. Java (JDK 8+)

. Swing (GUI Framework)

. AWT (Abstract Window Toolkit)

. Java I/O for piece images

## How to Play:

. Start the game - Run the main class

. Select a piece - Click on your piece (highlighted)

. Make a move - Click on a valid highlighted square

. Take turns - White moves first, then black alternately

. Win condition - Checkmate the opponent's king

## Controls:

. Click - Select pieces and make moves

. Game resets automatically after checkmate

## Requirements:

. Java JDK 8 or higher

. IDE (IntelliJ, Eclipse, or VS Code)

. Images folder with chess piece images
